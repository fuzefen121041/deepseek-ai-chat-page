# 媒体资源目录

这个目录用于存放视频和图片等静态资源。

## 目录结构

- `videos/` - 存放视频文件
- `images/` - 存放图片文件

## 使用说明

1. 将你的视频文件放在 `videos/` 目录下
2. 将你的图片文件放在 `images/` 目录下
3. 在 `src/data/mediaList.js` 中添加文件信息即可在媒体库页面显示

## 支持的文件格式

### 视频
- MP4 (.mp4)
- WebM (.webm)
- OGG (.ogg)

### 图片
- JPEG/JPG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- WebP (.webp)
